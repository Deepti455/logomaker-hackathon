# Logo Maker Hackathon

## Project Documentation

### Overview
Logo Maker is a collaborative project developed during the Hackathon. The aim of the project is to provide users with an intuitive interface to create logos effortlessly.

### Features
- User-friendly interface
- Customizable logo elements
- Download option for created logos
- Sharing options to social media

### Installation
To run the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Deepti455/logomaker-hackathon.git
   ```
2. Navigate to the project directory:
   ```bash
   cd logomaker-hackathon
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

### Usage
1. Open your browser and navigate to `http://localhost:3000`.
2. Use the interface to create and customize your logo.
3. Once finished, download or share your logo.
